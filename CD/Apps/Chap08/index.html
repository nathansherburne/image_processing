Java Applications

Chapter 8: Frequency Domain

ButterworthHighPass Applies a Butterworth high pass filter to an image.
ButterworthLowPass Applies a Butterworth low pass filter to an image.
IdealHighPass Applies an ideal high pass filter to an image.
IdealLowPass Applies an ideal low pass filter to an image.
MakeSinusoid Generates synthetic images in which grey level has a sinusoidal variation horizontally, vertically or in both directions.
SpectralProbe Displays an image and the spectrum of a small square region of that image, defined by the user.
Spectrum Computes the amplitude spectrum of an image, windowing if required, and writes it to a file as an image.
SpectrumViewer Displays an image and its spectrum side-by-side.
